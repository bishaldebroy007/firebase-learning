# Firebase

In this repo, I have kept all my practice materials for Firebase throughout my learning journey. 

## Installation and Setup

Use the package manager [yarn](https://yarnpkg.com/) to install.

```
yarn
```
***Or,***
```
yarn install
```
**Run the project:**
```
yarn dev
```
